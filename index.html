Mesh WiFi iPerf3 Throughput Data — July 2026

Dataset compiled by David Park | 175 measurements | 7 systems × 5 distances × 5 runs | July 13, 2026

This page documents iPerf3 v3.16 throughput measurements for 7 consumer mesh WiFi systems tested in a 3,500 sq ft two-story residential environment with drywall construction. Each reported value is the average of 5 independent iPerf3 runs (4 parallel TCP streams, 30-second duration) to a wired local server running Ubuntu 24.04 LTS. Client hardware: Intel BE200 WiFi 7 adapter on Windows 11 Pro 24H2. ISP connection: 2 Gbps fiber. Tests conducted April through July 2026, 02:00–05:00 local time.

Systems were selected in April 2026. WiFi 7 products including TP-Link Deco BE63/BE65 and Netgear Orbi 770 reached broad availability after testing began.
